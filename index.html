DATA PATH

Tujuan Pembelajaran

Setelah menggunakan media ini, pengguna diharapkan mampu:

01
Memahami konsep dasar alur data dalam sebuah sistem komputer mulai dari input hingga output.
02
Menjelaskan perbedaan dan fungsi antara perangkat input, pengolahan data, dan perangkat output.
03
Menyebutkan contoh nyata perangkat yang berperan dalam setiap tahapan pemrosesan data.
04
Mensimulasikan proses alur data secara interaktif untuk memperkuat pemahaman konseptual.
05
Mengevaluasi pemahaman melalui kuis interaktif dan mampu mengidentifikasi jawaban yang benar.
DATA PATH

Panduan Penggunaan

Ikuti langkah berikut agar bisa menggunakan media ini dengan optimal.

🏠

Langkah 1 – Mulai dari Halaman Utama

Tekan tombol MULAI pada halaman pembuka untuk masuk ke Menu Utama.

🗂️

Langkah 2 – Pilih Menu

Pilih salah satu menu yang tersedia: Tujuan Pembelajaran, Panduan, Materi, Kuis, atau Tentang.

📚

Langkah 3 – Pelajari Materi

Buka menu Materi, pilih topik yang ingin dipelajari: Input Data, Pengolahan Data, Hasil Data, atau Simulasi Data.

⚡

Langkah 4 – Coba Simulasi

Masukkan data pada halaman Simulasi Data untuk melihat secara langsung bagaimana data diproses dari input hingga output.

📝

Langkah 5 – Kerjakan Kuis

Setelah mempelajari materi, buka menu Kuis dan jawab 4 pertanyaan pilihan ganda. Jika salah, kamu bisa mengulang.

🔄

Langkah 6 – Navigasi

Gunakan tombol ← Kembali atau Menu di pojok atas untuk berpindah halaman kapan saja.

Tentang Media

TENTANG MEDIA

DATA PATH (Perjalanan Data dari Input ke Output) adalah media pembelajaran interaktif yang dirancang untuk membantu pengguna memahami bagaimana data diproses dalam sebuah sistem komputer secara bertahap. Media ini menyajikan konsep alur data mulai dari data masuk, diolah, hingga menghasilkan keluaran dengan pendekatan visual dan interaktif.
